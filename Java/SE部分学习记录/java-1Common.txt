1.JAVA：
①平台（字节码，JVM虚拟机）
②语言：和操作系统交互的媒介

2.JAVA语言版本：
①java ME（微型版，可以做一些终端的移动设备）
②*java SE（标准版，整个java语言的核心，理论部分，可以做一些桌面应用以及
③java EE（企业版，java优越性的体现，对SE封装的应用）

3.JAVA语言的特点：
①※跨平台性：依靠JVM实现，通过其特有的机制解决异平台差异的问题。
②支持多线程编程：
③自动垃圾回收机制：（两大核心机制之一）・
④健壮性：异常处理机制（两大核心机制之一）

4.JAVA环境：
①jdk(包含开发人员所用到的开发工具包） 
②jre(运行环境）{
核心类库（API） +  java虚拟机（JVM）
}

5.JAVA的数据类型：
①基本数据类型：（1字节=8bit=8位）
      a.整型(byte short int long)(1 2 4 8字节)
      b.浮点型(float double)（4 8字节） 
      c.字符型(char)（2字节）              
      d.布尔型（boolean)（1字节）
②引用数据类型：
      a.数组
      b.API
      c.自定义类
